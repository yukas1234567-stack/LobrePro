# Lobre Pro - Neon Block Blast

Bağımlılık yapan neon blok kırma oyunu. Combo, Mağaza, Klasik ve Zaman modlarıyla oynanabilir.

## Özellikler

- 🎮 Çoklu oyun modu (Combo, Mağaza, Klasik, Zaman Saldırısı)
- 📱 Ana ekrana eklenebilir PWA uygulaması
- 🌐 Çevrimdışı oyun desteği
- ✨ Neon efektleri ve animasyonlar
- 🎯 Türkçe arayüz

## Nasıl Oynanır

1. Bağlantıyı telefonunuzda açın
2. Safari (iOS) veya Chrome (Android) tarayıcısında "Ana Ekrana Ekle" seçeneğini seçin
3. Oyuna başlayın!

## Teknoloji

- HTML5 Canvas
- CSS3 Animasyonlar
- Service Worker (Çevrimdışı Destek)
- PWA Manifest
